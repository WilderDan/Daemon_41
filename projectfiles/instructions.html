    The Volkswagen Diesel Emissions Measurement Simulator shows test results for the Jetta and Passat. The outputs given by the simulation are based on several variables within a tolerance level. Our website demonstrates the levels given in a controlled lab test, as well as a real road driving test. The site itself draws no conclusions and intends for the user to run simulations to analyze the results.

Requirements:

  • Java
  • Javascript enabled
  • Chrome
    (Note: Other web browsers may work but have not been tested at this time.)

Label Descriptions

  1. Test Type - Selected these options to select the type of test to run
    • Custom Test - runs a simulated road emissions test with user selected traits.
    • Road Test - runs a preset test simulating road conditions
    • Lab Test - runs a standard lab condition emissions test

  2. Vehicle - Select a vehicle to test
    • Jetta
    • Passat

  3. Custom Road Test Options - User selected constraint (Only available on custom road test)
    • Distance
      1. Short - Short drive below 10 miles
      2. Medium - Moderate drive between 10-25 miles
      3. Long - Long drive over 25 miles
    • Terrain
      1. Straight - simulates output for a flat generally straight roads.
      2. Mixed - simulates output for a randomly simulated curvy or straight roads.
      3. Curvy - simulates output for a roadways with many sharp or large turns, and very few straightaways.
    • Driving Style
      1. Casual - Set test to run for casual driving situations (i.e. slow acceleration, low rpms, moderate speed,etc.).
      2. Normal - Set test to run for standard driving. No abnormal peaks or lulls in driving condition.
      3. Aggressive - Set test to run accounting for aggressive behavior (i.e. over-rev Engine, high speed,etc.).
    • Cold Start
      1. Yes - Runs a custom road test assuming the engine has not been prestarted.
      2. No - Runs the test with the enginie already warmed up.

  4. Run Test button

  5. Results Display (Visible once a test has been run)
    • NMOG (Non-Methane Organic Gases) - Measured in grams per mile
    • CO (Carbon Monoxide) - Measured in grams per mile
    • NOX (Nitrogen Oxide) - Measured in grams per mile
    • HCHO (Formaldehyde) - Measured in grams per mile
    • MPG (Miles Per Gallon)- Gas usage per mile based on test parameters.
    • Distance - Measured in miles
    • Average Speed - Measured in miles per hour

Instructions to Run a Test

1. First, choose a method of test to run: Custom Road Test, Road Test, or Lab Test.


Road test and Lab test will use preset constraints for the test.
Custom Road Test will allow the user to change certain constraints to allow for driving conditions.


2. Choose the type of vehicle to be used in the test. At this time only the VW Jetta and Passat are available.

3. (Custom Road Test ONLY) Adjust the Custom Test parameters


3a. Adjust the Distance (Custom Road Test ONLY)
Distance determines the approximate size of the sample for the test.
Also, after a certain distance/time, the simulation will enter warmed up engine state.


3b. Adjust the Road Type (Custom Road Test ONLY)
Road type affects the driving condition and consistency of the engine. Curvy roads require more acceleration and deceleration.


3c. Adjust the Driving Style (Custom Road Test ONLY)
Driving style affects the way the engine runs during the test. The casual state puts minimal effort on the engine while aggressive may accelerate fast and Over-Rev the engine.


3d. Adjust the Cold Start Test parameter (Custom Road Test ONLY)
Cold start affects whether the engine is treated as warmed up. Warmed up engines are able to operate slightly more efficiently than a cold engine. Cold start increase the quantity of gas emisssions until warm up.


4. Click the run button to run your test.

5. Review Results in Display. Results displayed: Each emission (NMOG,NOX,CO, and HCHO) in quantities of Grams per mile, miles per gallon average of the test, total distance of the simulation, and average speed of the simulation.
